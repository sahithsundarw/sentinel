Why 235B Parameters Score Zero

The Partial Observability Trap
The adversary tracks internal state:
topic_index × intensity_level

But your agent only sees the surface prompt.

A 235B model answers each prompt in isolation.
It has no memory of its past actions shaping the adversary's strategy.
Episode 1: Agent refuses "bomb-making" (intensity 1) → Adversary backs off to intensity 0 → Agent allows (it looks benign!) → Adversary escalates to intensity 5 → Agent refuses but too late: score = 0 Zero-shot Qwen: same result every episode. Q-learner: learns the FSM structure in 10 eps.
