SENTINEL LOCAL TRAINING — RTX 4060
===================================

STEP 1: Install dependencies (run once)
  pip install "unsloth[cu121-torch260] @ git+https://github.com/unslothai/unsloth.git"
  pip install trl datasets requests

STEP 2: Test GPU
  python -c "import torch; print(torch.cuda.get_device_name(0)); print(torch.cuda.get_device_properties(0).total_memory / 1e9, 'GB')"

STEP 3: Set HF token
  set HF_TOKEN=your_token_here   (Windows)

STEP 4: Run full training (SFT then PPO back to back)
  python scripts/train_local.py --phase all --episodes 20

STEP 5: Charts auto-update. Then commit:
  git add results/ checkpoints/ scripts/train_local.py generate_charts.py run_instructions.txt
  git commit -m "feat: local SFT+PPO training results (RTX 4060)"

EXPECTED TIME ON RTX 4060:
  SFT: ~45 minutes
  PPO (20 episodes): ~2-3 hours
  Total: ~3-4 hours

EXPECTED RESULTS:
  Zero-shot baseline: 0.5428
  After SFT:          ~0.75
  After PPO:          ~0.85

OPTIONAL — run phases separately:
  python scripts/train_local.py --phase sft        (SFT only)
  python scripts/train_local.py --phase ppo        (PPO only, needs SFT checkpoint)

RESULT FILES WRITTEN:
  results/llama_sft_scores.json    baseline_score, post_sft_score
  results/llama_ppo_scores.json    post_ppo_score, episode_rewards[]
  checkpoints/llama-sft/           LoRA SFT checkpoint
  checkpoints/llama-ppo/           LoRA PPO checkpoint
